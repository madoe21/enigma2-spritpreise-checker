SpritpreiseChecker

Enigma2 plugin — part of the madoe21 feed

Displays current fuel prices near you (Tankerkoenig API) on your Enigma2 receiver, sortable by price or distance.

Install

Add the feed once (skip if already done), then install this plugin — either from your receiver's plugin manager, or via SSH:

echo 'src/gz madoe21 https://madoe21.github.io/enigma2-madoe21-feed/feed' > /etc/opkg/madoe21-feed.conf
opkg update
opkg install enigma2-plugin-extensions-spritpreise-checker
